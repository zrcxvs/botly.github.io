About Botly

Botly is a Telegram bot creation and management service specifically designed for modern business needs. With cryptocurrency payment integration through NowPayments, Botly allows you to accept payments from around the world easily and securely.

Our Mission

To provide easy-to-use, reliable Telegram bot solutions integrated with modern payment systems to help businesses thrive in the digital era.

Advantages

Botly is built with Cloudflare Workers technology ensuring high speed and reliability, as well as scalability to handle thousands of users simultaneously.

Target Users

Perfect for Android platform beginners who want to automate their business without needing deep technical programming knowledge.

Our Services

Botly Workflow

1

User Interacts with Bot

Users send messages to the Telegram bot managed by Botly through BotFather.

2

Cloudflare Workers Process Requests

User requests are processed by Cloudflare Workers running on Cloudflare's global edge network.

3

Data Storage in Cloudflare KV

Order data and user sessions are stored in Cloudflare KV for fast and reliable access.

4

Payment Integration with NowPayments

For paid transactions, the system integrates with NowPayments to process cryptocurrency payments.

5

Notifications to Admin

Admin receives notifications via Telegram for every order or important activity.

Cloudflare Workers Configuration

Botly uses standardized configuration with secure environment variables:

KV Namespace

# KV Namespace configuration for data storage
[[kv_namespaces]]
binding = "ORDERS_KV"
id = "your_orders_kv_id"

[[kv_namespaces]]
binding = "SESSIONS_KV"
id = "your_sessions_kv_id"

Environment Variables

# Environment variables for bot configuration
[vars]
TELEGRAM_TOKEN = "your_telegram_bot_token"
ADMIN_CHAT_ID = "your_admin_chat_id"
NOWPAYMENTS_API_KEY = "your_nowpayments_api_key"
CURRENCY = "USD"
PAY_CURRENCY = "USDTBSC"
BASE_URL = "your_workers_dev_url"
Important Note: All actual tokens and API keys are configured through secure environment variables in the Cloudflare Dashboard and are never stored in publicly exposed code.
