Tentang Botly

Botly adalah layanan pembuatan dan pengelolaan bot Telegram yang dirancang khusus untuk kebutuhan bisnis modern. Dengan integrasi pembayaran kripto melalui NowPayments, Botly memungkinkan Anda menerima pembayaran dari seluruh dunia dengan mudah dan aman.

Misi Kami

Menyediakan solusi bot Telegram yang mudah digunakan, andal, dan terintegrasi dengan sistem pembayaran modern untuk membantu bisnis berkembang di era digital.

Keunggulan

Botly dibangun dengan teknologi Cloudflare Workers yang memastikan kecepatan dan keandalan tinggi, serta skalabilitas untuk menangani ribuan pengguna secara bersamaan.

Target Pengguna

Cocok untuk pemula di platform Android yang ingin mengotomatisasi bisnis mereka tanpa memerlukan pengetahuan pemrograman teknis yang mendalam.

Layanan Kami

Botly menawarkan berbagai layanan untuk memenuhi kebutuhan otomatisasi bisnis Anda:

Bot Pesan Otomatis

Buat bot yang dapat merespons pesan secara otomatis, memberikan informasi produk, dan menangani pertanyaan umum pelanggan 24/7.

Sistem Pemesanan

Kembangkan sistem pemesanan produk atau layanan melalui bot Telegram dengan proses yang mudah dan intuitif.

Integrasi Pembayaran Kripto

Hubungkan bot Anda dengan NowPayments untuk menerima pembayaran dalam berbagai kripto seperti USDT, Bitcoin, dan lainnya.

Manajemen Data Pelanggan

Simpan dan kelola data pesanan dan sesi pelanggan menggunakan Cloudflare KV untuk akses yang cepat dan aman.

Laporan dan Analitik

Sediakan dashboard sederhana untuk memantau aktivitas bot, pesanan, dan pembayaran yang masuk.

Dukungan dan Pemeliharaan

Berikan dukungan teknis berkelanjutan dan pemeliharaan rutin untuk memastikan bot Anda berjalan optimal.

Alur Kerja Botly

1

Pengguna Berinteraksi dengan Bot

Pengguna mengirim pesan ke bot Telegram yang dikelola oleh Botly melalui BotFather.

2

Cloudflare Workers Memproses Permintaan

Permintaan pengguna diproses oleh Cloudflare Workers yang berjalan di jaringan edge global Cloudflare.

3

Penyimpanan Data di Cloudflare KV

Data pesanan dan sesi pengguna disimpan di Cloudflare KV untuk akses yang cepat dan andal.

4

Integrasi Pembayaran dengan NowPayments

Untuk transaksi berbayar, sistem terintegrasi dengan NowPayments untuk memproses pembayaran kripto.

5

Notifikasi ke Admin

Admin menerima notifikasi melalui Telegram untuk setiap pesanan atau aktivitas penting.

Konfigurasi Cloudflare Workers

Botly menggunakan konfigurasi standar dengan variabel lingkungan yang aman:

KV Namespace

# Konfigurasi KV Namespace untuk penyimpanan data
[[kv_namespaces]]
binding = "ORDERS_KV"
id = "your_orders_kv_id"

[[kv_namespaces]]
binding = "SESSIONS_KV"
id = "your_sessions_kv_id"

Variabel Lingkungan

# Variabel lingkungan untuk konfigurasi bot
[vars]
TELEGRAM_TOKEN = "your_telegram_bot_token"
ADMIN_CHAT_ID = "your_admin_chat_id"
NOWPAYMENTS_API_KEY = "your_nowpayments_api_key"
CURRENCY = "USD"
PAY_CURRENCY = "USDTBSC"
BASE_URL = "your_workers_dev_url"
Catatan Penting: Semua token dan kunci API yang sebenarnya dikonfigurasi melalui variabel lingkungan yang aman di Cloudflare Dashboard dan tidak pernah disimpan dalam kode yang terpapar publik.
