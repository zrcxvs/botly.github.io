Tentang Botly

Botly adalah layanan pembuatan dan pengelolaan bot Telegram yang dirancang untuk kebutuhan bisnis modern.

Misi

Menyediakan solusi bot Telegram yang mudah digunakan dan andal.

Keunggulan

Dibangun dengan teknologi Cloudflare Workers untuk kecepatan dan keandalan tinggi.

Target Pengguna

Cocok untuk pemula di platform Android tanpa pengetahuan pemrograman teknis.

Layanan Kami

Bot Pesan Otomatis

Buat bot yang dapat merespons pesan secara otomatis.

Sistem Pemesanan

Kembangkan sistem pemesanan produk melalui bot Telegram.

Integrasi Pembayaran Kripto

Terima pembayaran dalam berbagai kripto melalui NowPayments.

Manajemen Data Pelanggan

Simpan dan kelola data pesanan menggunakan Cloudflare KV.

Teknologi yang Digunakan

Cloudflare Workers
Cloudflare KV
NowPayments API
Telegram Bot API

Konfigurasi Cloudflare Workers

KV Namespace

# Konfigurasi KV Namespace
[[kv_namespaces]]
binding = "ORDERS_KV"
id = "your_orders_kv_id"

[[kv_namespaces]]
binding = "SESSIONS_KV"
id = "your_sessions_kv_id"

Variabel Lingkungan

# Variabel lingkungan untuk konfigurasi bot
[vars]
TELEGRAM_TOKEN = "your_telegram_bot_token"
ADMIN_CHAT_ID = "your_admin_chat_id"
NOWPAYMENTS_API_KEY = "your_nowpayments_api_key"
CURRENCY = "USD"
PAY_CURRENCY = "USDTBSC"
BASE_URL = "your_workers_dev_url"
